### Sitemap::DB

## DESCRIPTION

This is a module for loading sitemap data into a database.

## INSTALLATION

To install this module, run the following commands:

	perl Makefile.PL
	make
	make test
	make install

## SUPPORT AND DOCUMENTATION

After installing, you can find documentation for this module with the
perldoc command.

    perldoc Sitemap::DB
